# Distributed Text Processing System

## Description
This project implements a distributed text processing system using TCP sockets.

## Features
- Distributed processing using multiple workers
- Multi-threading in coordinator 
- Word count, line count
- Word frequency analysis
- Top 5 most frequent words 
- Processing time measurement 

## How to Run

### 1. Start workers
python worker.py 9001
python worker.py 9002

### 2. Start coordinator
python coordinator.py

### 3. Run client
python client.py
